Welcome to my page

My summer project is about healthcare services.

The Goal

The goal of this project is to create a "diagnosis database". This is a database in which symptoms a patient is feeling can be entered and a diagnosis will be given. Along with the diagnosis, a treatment is provided as well.

The Dream

One day I hope to successfully create and implement this project into the medical field. This dream database could not only speed up the diagnosis process but also reduce the number of wrongful diagnosis and treatment cases.
